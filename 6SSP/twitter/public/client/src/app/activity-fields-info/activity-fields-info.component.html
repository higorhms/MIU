To create a new Running Activity Record, please follow these steps:

Name: This field requires the name of the running activity. It should be concise, yet descriptive.
Examples include"Interval Run", "Long Run", etc.

Duration: The duration field is meant for an estimated time range that it would typically take to perform the running activity.
You can use time ranges like "20-40 minutes", "60+ minutes", etc.

Description: This field is for a brief explanation of what the running activity is.
It should include the nature of the activity and any particular features it may have.
For example, an interval run could be described as "Interval runs involve periods of intense effort interspersed with periods of moderate to light effort. This training method improves speed and cardiovascular fitness."
